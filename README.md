# 🛒 Análise de Supermercado

Pipeline ETL completo para análise de produtos de supermercado.

## 📋 Funcionalidades
- Importação de CSV para SQLite com Python
- Consultas SQL: preço médio por marca, produtos mais caros/baratos
- Exportação para Excel com tabela dinâmica e gráficos
- Dashboard com PROCV, SE e formatação condicional

## 🛠️ Tecnologias
- Python 3
- SQLite3
- CSV
- Excel / Google Planilhas

## 📁 Arquivos
| Arquivo | Descrição |
|---------|-----------|
| `supermercado.csv` | Dados brutos dos produtos |
| `importar.py` | Código Python para importar e consultar |
| `README.md` | Este arquivo |

## 🚀 Como usar
1. Execute: `python importar.py`
2. Abra o CSV no Excel para análises
